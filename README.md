# Wikipedia Web Scraper (3 Tables Project)

## Overview
This project scrapes structured data from a Wikipedia page containing multiple tables of US companies by revenue.

## What it does
- Extracts 3 different HTML tables using BeautifulSoup
- Cleans and structures data using Pandas
- Converts scraped data into CSV files

## Tech Stack
- Python
- BeautifulSoup
- Requests
- Pandas

## Output
- Companies → CSV file
- Private_Companies → CSV file
- Profit_Companies → CSV file

## What I learned
- HTML parsing with BeautifulSoup
- Working with multiple tables
- Data cleaning using Pandas
- Exporting structured datasets
